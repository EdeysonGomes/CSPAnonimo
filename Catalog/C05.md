# **Competency C05 Specification**

### Competency Title

    Write a Technical Report


### Textual Description (final)

This competency involves the ability to **produce clear, structured, and technically accurate reports** that document computational problem-solving processes, decisions and results. Learners must communicate problem context, modeling choices, implementation details, and validation outcomes using appropriate technical language and standardized report structures.

Learners demonstrate this competency by:

* Structuring reports according to academic or professional conventions.
* Describing computational models and assumptions precisely.
* Reporting testing procedures and results objectively.
* Justifying design decisions using evidence and logical reasoning.



### Knowledge Specification (final)

* Written Communication (FPK)


### Disposition Specification (final)

* Collaborative, Meticulous, Responsible


### Knowledge–Skill Pairing (final)

* **Written Communication (FPK) – Apply** → write, revise, and clarify technical content.
