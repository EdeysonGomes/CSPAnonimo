# **Competency C04 Specification**

### Competency Title

    Define Regular Expressions for Finite Automata


### Textual Description (final)

This competency involves the ability to **define and apply regular expressions that accurately represent the languages recognized by finite automata**. Learners must understand the formal correspondence between finite automata and regular expressions and use this relationship to construct, interpret, and validate equivalent representations.

Learners demonstrate this competency by:

* Analyzing finite automata to identify accepted patterns and languages.
* Constructing regular expressions that describe the same language.
* Translating between automata and regular expressions using standard notation.
* Verifying representational equivalence through systematic testing or formal reasoning.

  → reason about equivalence, validate correctness, justify representations.
